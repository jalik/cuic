Cuic.js - Common User Interface Components

Cuic.js is a library that provides various components used in dynamic UIs (like notifications, tooltips, tabs...). It was designed to be simple and easy to use, with convention over configuration pattern.

Cuic.Dialog

Dialogs are used to display things that needs interactions, for example a confirmation, or a login form.

Cuic.Grid

The grid is a good way to dispose components using a coordinate system, allowing to drag them.
